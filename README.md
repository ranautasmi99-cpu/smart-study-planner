# smart-study-planner
This project presents a Smart Study Planner, a Python-based system that helps students organize their study schedule intelligently using rule-based logic inspired by Artificial Intelligence concepts.


##  Problem
Students struggle to manage multiple subjects, deadlines, and study time efficiently.

##  Solution
This project is a Python-based Smart Study Planner that prioritizes subjects based on deadline and difficulty.

##  Features
- Priority-based scheduling
- Intelligent study recommendations
- Simple and user-friendly

##  Concept Used
Rule-based AI (no machine learning)

##  How to Run
1. Open terminal
2. Run:
   python study_planner.py
3. Enter subject details

##  Tech Used
- Python

##  Output
Generates a personalized study plan with time recommendations.
